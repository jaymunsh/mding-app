## Notes

The app now ships as static PWA assets. Hosting is only needed to serve the app shell and updates over HTTPS; workspace data stays in browser-local IndexedDB and can be imported/exported.

On macOS, installed Chromium PWAs can register Markdown file handlers through the web app manifest. Safari-installed web apps do not currently provide this same file-handler path.
