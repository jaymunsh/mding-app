# mding PWA Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the SwiftUI harness with a local-first React TypeScript PWA that runs on iOS, iPadOS, and macOS.

**Architecture:** Vite builds a static installable PWA. Domain logic is pure TypeScript, IndexedDB owns persistent workspace data, React composes a responsive file tree and document editor using the tokens in `DESIGN.md`.

**Tech Stack:** Vite, React, TypeScript, Vitest, Biome, vite-plugin-pwa, idb, marked, lucide-react.
