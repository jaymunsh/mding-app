# Changelog

## 1.6.1 - 2026-07-18

- Confirm selected-item and full-workspace deletion before removing local data.
- Keep deleted items available to Undo for five seconds.
- Increase the pinned section background and divider contrast.

## 1.6.0 - 2026-07-18

- Pin Markdown files as shortcuts without moving their workspace location.
- Create named Markdown files and folders with validation and automatic `.md` completion.
- Drag files to folders or the workspace root with destination feedback and success-only confirmation.
- Use focused reading for Markdown and HTML previews, with desktop and mobile exit controls.
- Restore deleted items for five seconds and preserve pinned metadata in workspace backups.
- Use the localized quick guide, responsive sidebar controls, and compact toolbar overflow.
- Clear every local workspace file and folder from Settings after an explicit confirmation, with a five-second Undo window.
- Release obsolete HTML iframes, Mermaid SVGs, search state, and asynchronous rendering work when switching documents.
- Share one Mermaid theme observer across diagrams while keeping renderer and service-worker caches available offline.

Full release: https://github.com/jaymunsh/mding-app/releases
