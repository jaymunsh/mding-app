# Changelog

## 1.6.0 - 2026-07-18

- Pin Markdown files as shortcuts without moving their workspace location.
- Create named Markdown files and folders with validation and automatic `.md` completion.
- Drag files to folders or the workspace root with destination feedback and success-only confirmation.
- Use focused reading for Markdown and HTML previews, with desktop and mobile exit controls.
- Restore deleted items for five seconds and preserve pinned metadata in workspace backups.
- Use the localized quick guide, responsive sidebar controls, and compact toolbar overflow.
