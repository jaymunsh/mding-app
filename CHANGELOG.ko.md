# 변경 로그

## 1.6.2 - 2026-07-18

- 설치된 앱 안의 업데이트 기록을 제거하고, 현재 언어에 맞는 GitHub 변경 로그를 외부 브라우저에서 엽니다.
- 앱의 새 업데이트 표시는 유지하면서 영문·한국어 변경 로그를 공식 릴리스 기록으로 사용합니다.
- 공개 저장소에서 더 이상 필요하지 않은 내부 계획·검증 산출물을 정리했습니다.

## 1.6.1 - 2026-07-18

- 선택 항목과 전체 작업공간을 삭제하기 전에 확인합니다.
- 삭제한 항목을 5초 동안 실행 취소로 복구할 수 있습니다.
- 더 진한 배경과 구분선으로 고정 영역을 명확하게 표시합니다.

## 1.6.0 - 2026-07-18

- Markdown 파일을 원래 위치에 유지하면서 바로가기로 고정합니다.
- 이름 검증과 `.md` 자동 완성을 지원하는 Markdown 파일·폴더 생성을 추가했습니다.
- 폴더나 작업공간 루트로 파일을 끌어 놓을 때 이동 대상을 표시하고, 저장 성공 후에만 완료 상태를 보여줍니다.
- 데스크톱과 모바일의 Markdown·HTML 미리보기에서 집중 읽기를 사용할 수 있습니다.
- 삭제한 항목을 5초 동안 복구하고, 고정 정보를 작업공간 백업에 보존합니다.
- 다국어 빠른 가이드, 반응형 사이드바와 작은 화면용 도구 모음 메뉴를 지원합니다.
- 설정에서 확인 후 전체 로컬 작업공간을 지우고 5초 동안 실행 취소할 수 있습니다.
- 문서 전환 시 오래된 HTML iframe, Mermaid SVG, 검색 상태와 비동기 렌더링 작업을 정리합니다.
- Mermaid 테마 감시자를 공유하면서 렌더러와 서비스 워커 캐시는 오프라인에서 사용할 수 있게 유지합니다.

## 1.5.0 - 2026-07-05

- Markdown 문서 안을 검색하고 결과 사이를 이동할 수 있습니다.
- 미리보기 배율, 읽기 진행률과 더 안정적인 HTML 미리보기 탐색을 지원합니다.
- 차분한 읽기 화면을 위해 문서 도구 모음과 미리보기 화면을 다듬었습니다.

## 1.2.0 - 2026-07-04

- 문서별 마지막 읽기 위치를 로컬에 기억합니다.
- Markdown·HTML 미리보기 스크롤과 모바일 제스처를 개선했습니다.

## 1.0.0 - 2026-07-03

- 브라우저 로컬 작업공간에서 Markdown 파일을 만들고 편집할 수 있습니다.
- 읽기 전용 HTML 파일을 미리보고 표, 콜아웃, 코드, Mermaid가 포함된 Markdown을 렌더링합니다.
- 계정 없이 테마·언어 설정과 작업공간 백업 가져오기·내보내기를 사용할 수 있습니다.
